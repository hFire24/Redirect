Google Blocker Setup Instructions

1. Install a redirector extension

Install a redirector extension for your web browser of choice (mobile web browsers not supported). In this case, the exctension "Redirector" by Einar Egilsson was installed. It's available on Chrome, Firefox, and Opera.

2. Access redirector settings

Find the recently installed extension in the extensions menu, and press "Edit Redirects". Results may vary depending on the extension installed.

3. Create a new redirect

4. Add in the following information. You can do this for any search engine.

Description: Google Blocker
Example URL: https://www.google.com/search?q=tetris&
Include Pattern: *://www.google.*/search?q=*&*
Redirect to: $1://hfire24.github.io/Redirect/google-blocker?q=$3
Pattern Type: Wildcard

5. Hit "Save" and ensure the Google Blocker is enabled.

6. If you search for anything on Google, you will be redirected to the google-blocker page.

The same applies for every redirect of a search engine.

7. To disable the redirector, find the extension, and click "Disable Redirector".

Alternatively, you could go into "Edit Redirects" and disable one redirect.

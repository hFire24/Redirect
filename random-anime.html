Random Anime Instructions

  1. Pick a random Funimation anime (1 win).
  2. Pick a random Crunchyroll anime.
  3. Pick a random anime from your Plan to Watch (1 win).
  4. Watch all three simultaneously.

Random Funimation Anime

  1. Apply the following filters: Rating (TV-14, TV-PG, TV-G)
  2. Randomly generate a number between 1 and 559.
  3. Go to the Nth anime on this list, where N is the random number generated.
  4. Don't watch any live action anime or the ZZZ test. Generate another number instead.
  5. If you already watched that anime before 2021, rewatch it!
  6. Pick a random Crunchyroll anime.

Random Crunchyroll Anime

  1. Don't watch anything with a like:dislike ratio worse than 10:1.
  2. If you already watched that anime before 2021, rewatch it!
  3. Pick a random anime from your Plan to Watch.

Random Plan to Watch Anime

  1. Open your Excel Spreadsheet.
  2. Open the Plan to Watch Tab.
  3. Apply the following filters: Plan (Crunchyroll, Funimation, Hulu), Episodes (Less than 26), Priority (True)
  4. Optional: Set SFW to True if the other random anime aren't safe.
  5. Sort by Random.
  6. Select the first anime on this list. Ignore all future anime.
  7. Watch all three random anime.

Final Step

  1. Arrange your windows so that all three anime are displaying.
  2. Start playing all three anime simultaneously.
  3. The anime that generates the first laugh or grabs the most attention wins.
